The R227 intersecting-fiber D8-coset shadow is now available in
`root_cell_permutation_sat.py` via `--intersecting-coset-cuts`, and in the
24-representative suite via `root_cell_triangle_rep_cloud_suite.py

  - introduces exact relative-permutation literals for each intersecting pair
    and common disjoint fiber;
  - ties each relative permutation to the original S4 block-entry variables by
    Tseitin equivalence;
  - introduces right-D8-coset literals as ORs of the eight relative permutations
    in that coset;
  - forbids the `729 - 182 = 547` impossible coset patterns for each of the
    `105` intersecting rooted fiber pairs.
- `root_cell_intersecting_coset_sat_audit.py`
  - checks the duplicated SAT helper against the CP-SAT source projection;
  - verifies relative-permutation orientation over all `24 x 24` block pairs;
  - verifies the D8 cosets partition S4 as `8,8,8`.
- `root_cell_triangle_rep_cloud_suite.py --intersecting-coset-cuts`
  - exports or smoke-solves the same R220 representative suite with the new cuts.
