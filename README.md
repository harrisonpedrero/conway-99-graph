# Conway 99-Graph R230 Review Bundle

This repository contains a consolidated adversarial-review bundle for the
R230 computational nonexistence certificate for `srg(99,14,1,2)`.

Start here:

- `adversarial-review-r230/README.md` for the review checklist.
- `adversarial-review-r230/latex/conway_99_r230_nonexistence.tex` for the
  LaTeX writeup.
- `adversarial-review-r230/reports/FINAL_REPORT_R230_NONEXISTENCE.md` for the
  concise result report.

Large CNF/DRAT proof bodies are intentionally not committed to normal Git
history because several DRAT proofs exceed GitHub's 100 MB file limit.  Their
hashes, solver/checker logs, audit JSON, and regeneration/replay commands are
included in the bundle.
